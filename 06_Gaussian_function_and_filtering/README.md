# 가우시안 함수와 필터링

* 다운로드하기

git clone https://github.com/jetsonworld/OpenCV_On_JetsonNano.git

```
cd OpenCV_On_JetsonNano/05_Image_Filtering
g++ -ggdb averaging.cpp -o averaging `pkg-config --cflags --libs opencv4`
./averaging
```
